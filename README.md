## Tech Stack

- Vue.js
- Supabase
- Tailwind CSS


## Project Setup

```sh
npm install
```

```sh
npm run dev
```

## Test

```sh
npm run test:unit
```

## Supabase

```sh
npx supabase start
```

```sh
npx supabase db reset
```

```sh
npx supabase status
```

## Create Migration

```sh
npx supabase db diff --use-migra -f {migration_name}
```
